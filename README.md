# 🍲 Ghana Food Personality Quiz

**Created by: Kingsford** 🇬🇭

Answer 3 fun questions and discover which Ghanaian food matches your personality! 
Banku, Waakye, Fufu, Kelewele - which one are you?

## How to Run
1. Make sure you have Python 3 installed
2. Download `food_quiz.py` 
3. Run: `python food_quiz.py` in terminal

## Features
- Fun personality quiz 
- Colorful terminal output
- Screenshot & share results
